# Task 3 — Responsive Landing Page with Modern CSS

## Overview

In this task, we transformed a basic HTML landing page into a modern, responsive user interface using advanced CSS techniques.

The goal was to build a clean, scalable, and professional layout while following modern frontend best practices.

## Final Questions

### Why are we using clamp instead of media queries?

Clamp allows fluid typography that automatically adapts to screen size without needing multiple media queries, making the code simpler and more maintainable.

---

### Why did we use minmax instead of fixed columns?

minmax creates flexible layouts that adapt to screen size, unlike fixed columns which can break on smaller devices.

---

### Why is mobile-first important?

Mobile-first improves performance, simplifies design, and ensures better usability on smaller devices before scaling up.

---

### What happens if we remove CSS variables?

Without variables, the design becomes harder to maintain and inconsistent, since changes must be made in multiple places instead of one.

---
